### 📍 LocalLens *(Flagship Project — Active)*
> A community-driven geo-discovery platform. Real places. Real people. No algorithmic noise.

**Key Features:**
- 🗺️ Interactive Leaflet map with geo-tagged posts
- 📸 Photo uploads pinned to real locations
- 🔥 Trending spots near you
- 👤 Local profiles & following
- 🏷️ Rich categories (Food, Culture, Nature, Urban, etc.)

**Traction:** 500+ spots • 15K+ views • 200+ organic users

**Stack:** `Laravel` • `React` • `Inertia.js` • `Tailwind` • `PostGIS` • `Leaflet`
